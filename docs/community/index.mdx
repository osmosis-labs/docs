# Community

Ways to contribute to Osmosis and participate in its ecosystem.

**Contributing** covers how to work with the Osmosis development repositories: issues, branching, and the contribution process. **Frontend Translations** is the guide for translating the Osmosis interface into new languages. The **IBC Relayer List** is a directory of relaying operators connecting Osmosis to the wider Cosmos ecosystem.
