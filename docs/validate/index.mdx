# Run a Node or Validator

This section is for node operators and validators: everything needed to install the `osmosisd` binary, sync a node, and run a validator on mainnet or testnet.

Start by [installing osmosisd](/build/developer-environment/osmosisd) (the binary's minimum specs and installation live under Build, since the same binary serves developers and operators), then follow **Running a Node** on testnet or mainnet to sync. For production key security before going live, see **Using TMKMS**, and for diagnosing a running node, see **Performance & Profiling**. To validate, **Validating on Testnet** and **Validating on Mainnet** cover creating a validator and going live.

Operators running IBC relaying infrastructure will also find the **Relayer Guide** (Hermes setup) here.
