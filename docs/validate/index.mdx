# Run a Node or Validator

This section is for node operators and validators: everything needed to install the `osmosisd` binary, sync a node, and run a validator on mainnet or testnet.

Start with **Osmosisd** for the binary's minimum specs and installation, then follow **Running a Node** for mainnet or testnet to sync. To validate, **Validating on Mainnet** and **Validating on Testnet** cover creating a validator, key management, and going live. For production key security, see **Using TMKMS**, and for diagnosing a running node, see **Performance & Profiling**.

Operators running IBC relaying infrastructure will also find the **Relayer Guide** (Hermes setup) here.
